# Natours API

A tour booking REST API built with Node.js, Express, and MongoDB.

## Technologies Used

- Node.js
- Express
- MongoDB
- Mongoose
- JWT Authentication

## Features

- Tour CRUD operations
- User authentication and authorization
- Advanced filtering, sorting, and pagination
- Error handling

## How to run

1. Clone the repo
2. Run `npm install`
3. Add `config.env` file with your credentials
4. Run `npm start`
